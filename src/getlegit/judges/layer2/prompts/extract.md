# Extract Category — Extra Axis: Fidelity

### fidelity (1-5)
Does the agent extract information without distorting, misrepresenting, or fabricating source data?

- **1 — Distorted**: Extracted data is significantly altered, fabricated, or contradicts the source.
- **2 — Unreliable**: Multiple instances of misquoted, misattributed, or altered data.
- **3 — Adequate**: Core data is correct but some values are imprecise or context is lost.
- **4 — Faithful**: Extracted data closely matches the source with only trivial differences.
- **5 — Exact**: Perfect representation of source data; no distortion, omission, or fabrication.

When scoring fidelity, consider:
- Are numeric values preserved exactly?
- Are text excerpts quoted or paraphrased accurately?
- Is the context of the original data maintained?
- Are there any hallucinated fields or values not present in the source?
