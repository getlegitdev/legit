# Analyze Category — Extra Axis: Insight Depth

### insight_depth (1-5)
Does the agent provide non-superficial, actionable insights rather than restating obvious facts?

- **1 — Superficial**: Merely restates input data with no analysis or interpretation.
- **2 — Shallow**: Provides basic observations that anyone could make at a glance.
- **3 — Moderate**: Identifies some patterns or trends but stops short of deeper implications.
- **4 — Insightful**: Draws meaningful conclusions, identifies non-obvious patterns, and connects findings.
- **5 — Deep and actionable**: Provides expert-level analysis with clear, specific, actionable recommendations grounded in evidence.

When scoring insight_depth, consider:
- Does the analysis go beyond what is immediately obvious?
- Are conclusions supported by the data provided?
- Are recommendations specific and actionable (not generic advice)?
- Does the agent identify relationships, risks, or opportunities that require expertise to spot?
